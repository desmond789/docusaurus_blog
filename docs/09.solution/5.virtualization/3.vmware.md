## **物理机不能访问虚拟机的解决方法**

**背景：** vmware station上有一臺過時的centos6，挂起再恢復后，xshell就無法鏈接該實例了，

實例無法ping通網關，外網能夠ping通網關，但是無法ping通實例。

**解決方案：** 到windows控制面板的網絡驅動器界面，重啓vmware相關的網絡驅動器即可恢復正常
