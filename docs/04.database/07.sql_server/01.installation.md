## 启动和连接数据库

### 启动数据库

有多种方法：

1. 命令行启动

   ```
   net start mssqlserver	#需要管理员权限
   ```
2. SQL SERVER配置管理器启动
   右击SQL Server（MSSQLSERVER）启动服务

   ![1714005931853](images/1714005931853.png)
3. Windows服务器启动

   ![1714006098888](images/1714006098888.png)

### 连接数据库

1. 从SQL SERVER Managment Studio，使用工具连接到数据库。
2. 身份验证可以选择：
   1. Windows身份验证
   2. SQL SERVER身份验证
