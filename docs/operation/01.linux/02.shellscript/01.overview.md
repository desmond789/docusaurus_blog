# 简介

Linux shell（或称为命令行界面或终端）是与操作系统内核通信的用户界面。它允许用户通过文本命令来执行各种任务，包括文件管理、进程控制、软件安装等。在Linux系统中，常见的shell包括Bash（Bourne Again SHell）、Zsh（Z Shell）、Fish等
